# hlpr website

A static, single-page website for hlpr.

## Files

- `index.html` — the website
- `video.mp4` — add your video using this exact filename

The video is responsive, preserves its original aspect ratio, and will never display wider than 720 pixels.

## Publish

Upload `index.html` and your `video.mp4` file to the root of the same GitHub repository.
