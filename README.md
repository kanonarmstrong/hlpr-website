# hlpr website

A static, single-page website for hlpr.

## Open locally

Double-click `index.html` or drag it into a browser.

## Publish

Upload `index.html` to any static host, including Netlify, Vercel, GitHub Pages, Cloudflare Pages, or an S3-compatible host.

## Customize

Search for `support@hlprcorp.com` in `index.html` to change the support address.
